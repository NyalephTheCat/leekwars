# E0273: included file unreadable

An `include("name")` statement resolved to a file, but the file
could not be read — for example because of filesystem permissions or
because its contents are not valid UTF-8. The diagnostic message
carries the underlying I/O error:

```text
error[E0273]: included file `utils` is unreadable: permission denied
```

Fix the file itself: make it readable by the toolchain and ensure it
is UTF-8 encoded text.

This is distinct from E0272 (the name didn't resolve to any file at
all) and E0274 (the file was read but failed to parse).
