# E0242: class member does not exist

An object member access names a field or method the class does not
declare (nor inherit). Inside a class this typically means
`this.something` where `something` was never declared as a field; it
also fires when accessing a member on a value known to be `null`.

Erroneous example:

```leekscript
class Leek {
    life = 100
}

var l = new Leek()
debug(l.hp)             // class `Leek` has no member `hp`
```

Use the member that exists, or declare the missing one:

```leekscript
class Leek {
    life = 100
}

var l = new Leek()
debug(l.life)
```

The check walks the inheritance chain, so members of a parent class
are found. Note that fields must be *declared* in the class body —
LeekScript objects don't grow new fields on assignment the way plain
maps do. If you need open-ended keys, use a map and `m["key"]`
indexing instead of an object.
