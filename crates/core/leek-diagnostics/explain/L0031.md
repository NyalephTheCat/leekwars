# L0031: approximated constant

A real literal spells out an approximation of a well-known constant
such as π or e. The builtin constant is both more precise and
self-documenting — `PI * r * r` says what the formula means, while
`3.14159` quietly loses digits.

Erroneous example:

```leekscript
var area = 3.14159 * r * r
```

Use the builtin constant:

```leekscript
var area = PI * r * r
```

Detection requires the literal to be at least 4 characters and a
digit-for-digit prefix of the real constant — `3.1` stays quiet (too
short to be clearly π), while `3.14`, `3.1416`, and a fully spelled
value all fire. A literal that *differs* from the constant's digits
(say `3.15`) is never flagged. This is a *pedantic* lint: it only runs
with `miku lint --pedantic` or `pedantic = true` under `[lint]` in
Miku.toml. Silence with `// @allow(L0031)`.
