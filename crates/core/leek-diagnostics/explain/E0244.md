# E0244: class static member does not exist

A static member access (`ClassName.member`) names a static field or
static method the class does not declare or inherit.

Erroneous example:

```leekscript
class Registry {
    static total = 0
}

debug(Registry.count)   // class `Registry` has no static member `count`
```

Use the static member that exists, or declare the missing one:

```leekscript
class Registry {
    static total = 0
}

debug(Registry.total)
```

Static lookup is distinct from instance lookup: `Registry.total`
reads a static field, while `new Registry().total` would need an
*instance* field (and is reported as E0242 if absent). A handful of
reflection intrinsics are always available on classes — `name`,
`fields`, `methods`, `parent`, `class`, and friends — and are never
flagged by this check.
