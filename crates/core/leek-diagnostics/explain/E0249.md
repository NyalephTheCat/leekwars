# E0249: protected static method

A static method declared `protected` is called from outside the
class hierarchy that owns it. A protected static method may be
called by the declaring class and its subclasses, but not by
unrelated code.

Erroneous example:

```leekscript
class Base {
    protected static seed() {
        return randInt(0, 100)
    }
}

var s = Base.seed()     // static method `seed` is protected
```

Call it from within the hierarchy, or make it public:

```leekscript
class Base {
    protected static seed() {
        return randInt(0, 100)
    }
}

class Roller extends Base {
    static roll() {
        return Base.seed()      // fine: subclass access
    }
}

var s = Roller.roll()
```

See E0248 for the `private static` variant and E0247 for protected
*instance* methods.
