# E0257: unknown interface

A class's `implements` clause names an interface that isn't declared
in the sources being checked. This is a warning rather than an error
because the interface may legitimately live in a file the checker
cannot see.

Example:

```leekscript
class Dog implements Nameable {   // unknown interface `Nameable`
    string name = "rex"
}
```

Declare the interface (or fix the spelling):

```leekscript
interface Nameable {
    string name;
}

class Dog implements Nameable {
    string name = "rex"
}
```

When the interface *is* known, its members are checked against the
class and missing ones are reported as E0256. Interfaces are an
experimental feature (enabled with the `LEEK_EXPERIMENTAL_INTERFACES`
environment variable).
