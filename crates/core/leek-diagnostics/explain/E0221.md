# E0221: function not available

A builtin function exists, but not in this file's LeekScript version —
it was introduced later than the `@version` the file declares. The
call would compile under a newer version, so the fix is usually a
version bump rather than a rename.

Erroneous example:

```leekscript
// @version:3
var found = mapContainsKey(data, "fire")   // not available at @version:3
```

Raise the file's version (and migrate anything the new version
changes):

```leekscript
// @version:4
var found = mapContainsKey(data, "fire")
```

`miku migrate` can apply known version-to-version rewrites
automatically. The mirror-image error is E0261: a function that
existed in an *older* version and has since been removed.
