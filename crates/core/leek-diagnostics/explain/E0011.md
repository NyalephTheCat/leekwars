# E0011: bad version pragma

A `@version` pragma is missing its value or names a version that
doesn't exist. LeekScript versions are 1, 2, 3, and 4; anything else
(or a bare `// @version`) is rejected, and the file falls back to its
default version.

Erroneous example:

```leekscript
// @version:5
var x = 1
```

Name a real version:

```leekscript
// @version:4
var x = 1
```

The pragma must be a line comment of exactly the shape
`// @version:N`. The version matters from the very first token — it
decides which words are keywords (`class` is only a keyword from
version 3, for instance) — which is why a bad value is an error
rather than a warning.
