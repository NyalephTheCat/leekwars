# E0264: protected static field

A static field declared `protected` is accessed from outside the
class hierarchy that owns it. A protected static field is visible to
the declaring class and its subclasses, but not to unrelated code.

Erroneous example:

```leekscript
class Settings {
    protected static verbose = false
}

debug(Settings.verbose)   // static field `verbose` is protected
```

Access it from within the hierarchy, or through a public accessor:

```leekscript
class Settings {
    protected static verbose = false

    static isVerbose() {
        return Settings.verbose
    }
}

debug(Settings.isVerbose())
```

See E0263 for the `private static` variant and E0243 for protected
*instance* fields.
