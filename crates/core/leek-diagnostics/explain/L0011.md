# L0011: self assignment

An assignment's target and value are the same side-effect-free place:
`x = x`, `this.life = this.life`. The statement does nothing — it is
either a leftover from an edit or a typo for a different source
variable.

Erroneous example:

```leekscript
var dmg = base
dmg = dmg          // does nothing
```

Remove the assignment, or assign the value you intended:

```leekscript
var dmg = base
dmg = dmg * multiplier
```

Compound assignments (`x += x`) are not flagged — doubling a value via
itself is meaningful. The diagnostic ships an autofix that removes the
statement. Silence an intentional case with `// @allow(L0011)`.
