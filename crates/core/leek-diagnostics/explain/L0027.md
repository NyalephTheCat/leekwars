# L0027: deep nesting

Control flow is nested more than 4 levels deep. Each level of
`if`/`for`/`while` multiplies the state a reader must hold in their
head to know when the innermost line runs.

Erroneous example:

```leekscript
for (var e in enemies) {
    if (isAlive(e)) {
        for (var w in getWeapons()) {
            if (lineOfSight(getCell(), getCell(e))) {
                while (getTP() >= 5) {   // five levels deep
                    useWeapon(e)
                }
            }
        }
    }
}
```

Early exits and extracted helpers flatten the pyramid:

```leekscript
for (var e in enemies) {
    if (!isAlive(e)) continue
    setWeapon(getWeapons()[0])
    useWeapon(e)
}
```

Only the statement that *opens* the offending level is reported —
anything nested deeper would be redundant noise. This is a *pedantic*
lint: it only runs with `miku lint --pedantic` or `pedantic = true`
under `[lint]` in Miku.toml. Silence with `// @allow(L0027)`.
