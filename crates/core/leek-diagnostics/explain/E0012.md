# E0012: invalid pragma value

A pragma that takes no value was given one. `@strict` is a flag — it
is either present or absent — so `// @strict:true` is rejected
rather than guessed at.

Erroneous example:

```leekscript
// @strict:true
var x = 1
```

Write the flag bare:

```leekscript
// @strict
var x = 1
```

The pragma grammar is `// @name` for flags and `// @name:value` for
valued pragmas (`@version:4`, `@experimental:feature`). Mixing the
two shapes up is the only way to hit this error today.
