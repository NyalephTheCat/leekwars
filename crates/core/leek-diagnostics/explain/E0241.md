# E0241: private method

A method declared `private` is called from outside the class that
owns it. Private methods are implementation details: only the
declaring class's own methods and constructors may call them.

Erroneous example:

```leekscript
class Strategy {
    private score(target) {
        return getLife(target)
    }
}

var s = new Strategy()
s.score(getNearestEnemy())   // method `score` is private
```

Call it through a public entry point, or widen its visibility:

```leekscript
class Strategy {
    pick() {
        return this.score(getNearestEnemy())
    }

    private score(target) {
        return getLife(target)
    }
}

var s = new Strategy()
s.pick()
```

Subclasses cannot call a parent's private methods either; use
`protected` when the method is part of the contract between a class
and its subclasses (see E0247).
