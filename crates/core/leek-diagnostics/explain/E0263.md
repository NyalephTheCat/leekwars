# E0263: private static field

A static field declared `private` is accessed from outside the class
that owns it. A private static field is readable and writable only
from inside the declaring class's own methods.

Erroneous example:

```leekscript
class Counter {
    private static total = 0
}

debug(Counter.total)    // static field `total` is private
```

Expose it through a public static method, or widen its visibility:

```leekscript
class Counter {
    private static total = 0

    static bump() {
        Counter.total++
        return Counter.total
    }
}

debug(Counter.bump())
```

Subclasses cannot read a parent's private static fields either; use
`protected static` when the field is part of the contract between a
class and its subclasses (see E0264).
