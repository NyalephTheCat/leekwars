# E0272: included file not found

An `include("name")` statement names a file that could not be
resolved to any source file of the project.

Erroneous example:

```leekscript
include("utls")         // included file `utls` not found
```

Check the spelling and the file's location:

```leekscript
include("utils")
```

The include name is given without the `.leek` extension and is
resolved relative to the including file's project. If the file
exists but cannot be read (permissions, encoding), the error is
E0273 instead; if it reads but fails to parse, see E0274.
