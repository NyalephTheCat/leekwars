# L0028: collapsible if

An `if` statement's entire body is another `if`, and neither has an
`else`. The two conditions can be joined with `&&`, saving a level of
nesting without changing behavior.

Erroneous example:

```leekscript
if (enemy != null) {
    if (getCellDistance(myCell, enemyCell) < 5) {
        useWeapon(enemy)
    }
}
```

Combine the conditions:

```leekscript
if (enemy != null && getCellDistance(myCell, enemyCell) < 5) {
    useWeapon(enemy)
}
```

If either `if` has an `else` branch, the rewrite would change meaning
and nothing is flagged. There is no autofix: joining conditions can
require parentheses (`a || b` and `c` must join as `(a || b) && c`),
so the rewrite is left to you. This is a *pedantic* lint: it only runs
with `miku lint --pedantic` or `pedantic = true` under `[lint]` in
Miku.toml. Silence with `// @allow(L0028)`.
