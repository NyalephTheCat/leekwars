# E0210: cannot assign value

The left-hand side of an assignment is not something that can be
assigned to. Assignable places are variables, fields, and indexed
elements (`x`, `obj.field`, `arr[i]`); literals, call results,
constants, and arbitrary expressions are not.

Erroneous example:

```leekscript
5 = x            // a literal is not a place
getCell() = 12   // a call result is not a place
PI = 3           // constants cannot be reassigned
```

Assign *to* a variable instead:

```leekscript
var c = getCell()
c = 12
```

This also fires for assigning to a class *method* through an instance
(`obj.method = …`) — methods are part of the class, not per-instance
data; use a field if you need a per-instance function value.
