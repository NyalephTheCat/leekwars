# E0247: protected method

A method declared `protected` is called from outside the class
hierarchy that owns it. Protected methods may be called by the
declaring class and its subclasses, but not by unrelated code.

Erroneous example:

```leekscript
class Unit {
    protected recover() {
        return getLife()
    }
}

var u = new Unit()
u.recover()             // method `recover` is protected on class `Unit`
```

Call it from a subclass, or expose a public wrapper:

```leekscript
class Unit {
    protected recover() {
        return getLife()
    }
}

class Healer extends Unit {
    turn() {
        return this.recover()   // fine: subclass access
    }
}

new Healer().turn()
```

Use `private` (E0241) to hide a method from subclasses too, or
`public` to open it to everyone.
