# L0025: too many arguments

A function, method, or constructor takes more than 5 parameters. Long
parameter lists are hard to call correctly — at the call site,
`fight(7, 3, true, false, 12, null)` gives no hint which `7` was the
damage and which the range, and transposed arguments compile fine.

Erroneous example:

```leekscript
function score(cell, dist, hp, tp, mp, threat) {
    // ...
}
```

Group related values into a map, object, or class:

```leekscript
class Situation {
    cell; dist; hp; tp; mp; threat;
}
function score(situation) {
    // ...
}
```

Lambdas are exempt — their arity is dictated by the caller. This is a
*pedantic* lint: it only runs with `miku lint --pedantic` or
`pedantic = true` under `[lint]` in Miku.toml. Silence a single case
with `// @allow(L0025)`.
