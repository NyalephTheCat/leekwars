# E0261: removed function

A builtin function existed in an older LeekScript version but was
removed before the version this file declares. The call can never
work at this version — the function is gone, usually replaced by a
renamed or redesigned equivalent.

Erroneous example:

```leekscript
// @version:4
removeKey(damages, "fire")      // removed in v4; use `mapRemove`
```

Use the current equivalent:

```leekscript
// @version:4
mapRemove(damages, "fire")
```

The diagnostic names the version the function was removed in. If you
are upgrading a script across versions, `miku migrate` applies the
known renames automatically; the deprecation lint (L0008) warns about
these functions *before* the version where they disappear. The
mirror-image error is E0221: a function that doesn't exist *yet* at
the file's version.
