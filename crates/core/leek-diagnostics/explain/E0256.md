# E0256: interface member missing

A class declares `implements Interface`, but doesn't provide every
field and method the interface requires (either directly or through
an ancestor class).

Erroneous example:

```leekscript
interface Named {
    string name;
    string describe();
}

class Dog implements Named {    // class `Dog` implements `Named`
    string name = "rex"         // but is missing method `describe()`
}
```

Provide the missing members:

```leekscript
interface Named {
    string name;
    string describe();
}

class Dog implements Named {
    string name = "rex"

    string describe() {
        return this.name
    }
}
```

Members inherited from a parent class satisfy the interface — only
truly absent members are listed. Interfaces are an experimental
feature (enabled with the `LEEK_EXPERIMENTAL_INTERFACES` environment
variable). If the interface name itself is unknown, the softer E0257
warning fires instead.
