# L0026: long function

A function or method body contains more than 60 statements, counted
recursively through nested blocks. Bodies that long usually do several
jobs at once — a fight loop that also scores cells, picks weapons, and
logs debug output is four functions in a trench coat — and each job is
harder to test and reuse while buried in the trench coat.

Example shape that triggers it:

```leekscript
function playTurn() {
    // 60+ statements of target selection, scoring,
    // movement, attacking, and logging ...
}
```

Extract cohesive chunks into named helpers:

```leekscript
function playTurn() {
    var target = getNearestEnemy()
    moveToward(target)
    useWeapon(target)
}
```

Statements are counted rather than lines, so formatting and blank
lines don't affect the threshold. This is a *pedantic* lint: it only
runs with `miku lint --pedantic` or `pedantic = true` under `[lint]`
in Miku.toml. Silence a deliberate case with `// @allow(L0026)`.
