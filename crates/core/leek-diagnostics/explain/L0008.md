# L0008: deprecated feature

A built-in function that still works in the current LeekScript version
has a recommended replacement, and is scheduled to disappear in a
later version. The call compiles and runs today, but migrating now
avoids a hard `removed function` error (E0261) after a version bump.

Erroneous example:

```leekscript
var r = randFloat(0, 1)        // deprecated; `randReal` replaces it
var s = subArray(arr, 0, 2)    // deprecated; `arraySlice` replaces it
```

Use the replacement named in the warning:

```leekscript
var r = randReal(0, 1)
var s = arraySlice(arr, 0, 2)
```

The deprecation table mirrors the official LeekScript versioning:
each entry warns in every version up to and including the one where
the old name was last available. The warning carries a suggested
rename, so `miku fix` can usually apply the migration automatically.
Silence a deliberate use with `// @allow(L0008)`.
