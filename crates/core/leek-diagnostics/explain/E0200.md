# E0200: unknown variable

A name is used that no declaration, parameter, builtin, or included
file provides — `unknown variable or function`. Most often it's a
typo of a nearby name, a variable used before any `var` declares it,
or a forgotten `include`.

Erroneous example:

```leekscript
var total = 0
totl += damage      // typo: `totl` was never declared
```

Fix the spelling (the diagnostic suggests an in-scope name when one
is close enough, and ships it as an autofix):

```leekscript
var total = 0
total += damage
```

If the name lives in another script of your AI, make sure that file
is pulled in with `include("...")` before the use. Globals must be
declared with `global` before they are visible.
