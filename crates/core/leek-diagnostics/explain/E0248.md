# E0248: private static method

A static method declared `private` is called from outside the class
that owns it. Visibility rules apply to static members exactly as
they do to instance members: a private static method is callable
only from inside the declaring class.

Erroneous example:

```leekscript
class MathKit {
    private static clamp(x, lo, hi) {
        return min(max(x, lo), hi)
    }
}

var v = MathKit.clamp(7, 0, 5)   // static method `clamp` is private
```

Route the call through a public static method, or widen visibility:

```leekscript
class MathKit {
    static clamp01(x) {
        return MathKit.clamp(x, 0, 1)
    }

    private static clamp(x, lo, hi) {
        return min(max(x, lo), hi)
    }
}

var v = MathKit.clamp01(7)
```

The protected variant — callable from subclasses as well — is
reported as E0249 when violated.
